# Purpose

This dashboard uses data downloaded from ESSENCE to track daily incidence of selected syndromes (diseases and other health problems) in Kansas City, Missouri. SaTScan software is used to detect spatial and/or temporal clusters of these syndromes.

# ESSENCE data

- Records of emergency room visits
- Missouri hospitals report within three business days
- Out of state hospitals report slower
- The data can be pulled by patient residence based on the ZIP code associated with the record or by hospital location.
    + Datasets pulled by patient residence consist of records in which the patient's ZIP code is at least partly within the Kansas City boundary. Therefore, a small number of records represent non-residents of Kansas City.
    + Datasets pulled by hospital location consist of records in which the hospital is within the Kansas City boundary. This captures all individuals, both residents and non-residents, seen at these hospitals.
- Data may contain duplicates. An effort is made to deduplicate, but there is a potential duplicate error rate (table).
